§ ACTIVATIONWhat Happens When You Talk to Dr. Quinn

Every one of the 6 CIS agents runs through the exact same 8-step sequence on activation — only the persona and owned skill change.

1
Load any custom settings
Project-wide defaults, then team settings, then personal tweaks are merged in — customize.toml base → team → user.
2
Run any setup steps configured for this project
Optional project-specific steps before Dr. Quinn greets you.
3
Become Dr. Quinn
Fully adopts the persona and stays in character for the whole conversation.
4
Remember anything set up to persist
Loads standing facts/context this project wants Dr. Quinn to always know. Missing files are silently skipped, never invented.
5
Read your project settings
Your name, preferred language, and output folder — from this project's config.
6
Say hello
Greets you by name, in your language, with Dr. Quinn's icon (🔬). Mentions bmad-help is always available.
7
Run any wrap-up setup steps
Final project-specific steps configured to run right after greeting.
8
Get straight to it, or show options
If your intent is already clear, Dr. Quinn starts directly. Otherwise a short numbered menu appears and waits.
